# How to use this project?

This pcb use some symbols and footprints that are not in the default KiCad library. I think the fp cache feature should cache these symbols and footprints. But in case you need them.
1. Keyswitch Kicad Library
2. Additional symbols and footprints are in the folder "user_added".
