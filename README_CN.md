# 如何使用本项目？

本 PCB 使用了一些不在 KiCad 默认库中的符号和封装。fp 缓存功能应该会自动缓存这些符号和封装，但如果你需要手动获取，可以参考以下来源：
1. Keyswitch KiCad 库
2. 额外的符号和封装位于 "user_added" 文件夹中。

## 3D 外壳设计

本项目包含键盘外壳的 FreeCAD 3D 模型文件：`keyboard-case.FCStd`。你可以使用 [FreeCAD](https://www.freecad.org/) 打开该文件，查看、修改或导出外壳设计，以便进行 3D 打印或 CNC 加工。
